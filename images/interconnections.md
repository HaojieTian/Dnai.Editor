## Interconnections

Notre solutions est composée de quatre parties principales. Elles même composées de sous parties.

- Core contient le core deamon (pour la partie algorithmique), un environnement python (pour le machine learning) ainsi qu'un server permettant le dialogue en TCP.
- Editeur contient l'interface du logiciel, ainsi qu'un updater permettant ça mise à jour.
- Web est constitué d'une API, d'un Hub (permettant l'échange des IA), ainsi qu'un site internet.
- Plugins, actuellement nous avons un plugin pour Unity3D
